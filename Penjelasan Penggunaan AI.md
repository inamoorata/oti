1. Letak Penggunaan AI
Penggunaan AI dilakukan ketika pengembangan dan debugging pada project ini.
Media AI yang Digunakan: Gemini AI
Aplikasi: Digunakan melalui chatbot untuk tanya jawab dan membantu analisis kode.
Kode yang Terlibat: Melibatkan source code HTML, CSS, dan JavaScript yang dikembangkan untuk project ini.

2. Tujuan dan Detail Penggunaan AI
Penggunaan AI bertujuan untuk sarana edukasi/pembelajaran dan Debugging.

a. Tujuan Utama: Pembelajaran dan Peningkatan Pemahaman Kode
Penggunaan AI sebagai media pembelajaran interaktif untuk memperkuat pemahaman mendalam tentang elemen dasar HTML, CSS, dan JavaScript.

Detail Penggunaan:
1. Bertanya mengenai kegunaan spesifik dari berbagai tag HTML, properti CSS (flex-direction, grid-template-columns, position, dll.), dan syntax JavaScript.
2. Meminta penjelasan detail mengenai konsep penting (misalnya, Box Model, Flexbox vs. Grid) dengan bahasa yang jelas dan mudah dipahami.
3. Mempelajari dan memverifikasi penggunaan tag HTML semantik yang tepat, seperti footer, header, nav, dan section.

b. Tujuan Kedua: Debugging dan Implementasi Fitur Khusus

AI digunakan untuk menganalisis potongan kode dan memberikan saran perbaikan untuk mencapai visual yang sesuai dengan desain, khususnya pada bagian-bagian yang kompleks.

Detail Penggunaan pada Masalah Spesifik:
1. Pengaturan Gambar: Meminta saran CSS (seperti penggunaan object-fit: cover; atau position: absolute; dan z-index) untuk mengecek apakah gambar dapat fit dan memenuhi frame kontainernya.
2. Layout Kustom (Footer & Kolom): Meminta bantuan dalam menyusun struktur CSS (menggunakan Flexbox atau Grid) untuk:
        -Mengatur tata letak kolom-kolom pada bagian footer.
        -Menata gambar pada dua kolom dengan ukuran lebar yang berbeda (misalnya, 1 kolom besar dan 1 kolom kecil) pada bagian portfolio.
3. Fitur Show More/Show Less (JS): Mencari dan memvalidasi logika JavaScript yang tepat untuk mengimplementasikan fungsionalitas show more/show less (toggle), termasuk event listener.
4. Pencarian Bug:Memberikan source code yang bermasalah, misalnya layout yang berantakan atau bug fungsional untuk dianalisis oleh AI untuk mendapatkan akar masalah dan solusi perbaikan.
